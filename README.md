# KodiPay

A rental management platform that helps landlords, tenants, and caretakers manage rental properties, payments, and maintenance requests with integrated M-Pesa payment support.

## Description

KodiPay simplifies rental management by providing a complete system for property management, tenant tracking, rent collection, and maintenance coordination. The platform supports multiple user roles and integrates with M-Pesa for seamless payment processing.

## Installation

1. Clone the repository:
```bash
git clone https://github.com/Johnnierad24/KodiPay.git
cd KodiPay/backend
```

2. Install dependencies:
```bash
npm install
```

3. Set up the database and configure environment variables as described in the project documentation.

4. Start the server:
```bash
npm start
```

## Usage

Once the server is running, the API will be available for integration with the frontend application. Refer to the source code for API endpoint details and integration guidelines.
